## 섹션 0: 5분 빠른 시작 (처음 사용자 필독)

> **목표**: Outlook을 처음 사용하는 분들이 5분 안에 기본 사용을 시작할 수 있도록 돕습니다.

### ✅ 체크리스트: 따라하기

#### 1단계: Outlook 접속하기 (1분)
**방법 A - 웹 브라우저 사용 (가장 쉬움)**
1. 웹 브라우저(크롬, 엣지 등) 실행
2. 주소창에 `outlook.office.com` 입력
3. 회사 계정 입력: `[본인 EP ID]@poscointl1.onmicrosoft.com`
4. 비밀번호 입력 후 로그인

[이미지: 로그인 화면]

**방법 B - PC 앱 사용**
1. 시작 메뉴에서 "Outlook" 검색
2. **Outlook (new)** 또는 **Outlook for Windows** 실행
3. 회사 계정으로 로그인

> **처음이라면?** 웹 브라우저 방법(방법 A)을 추천합니다. 설치가 필요 없고 어디서나 사용 가능합니다.

   - 좌측 상단의 **새 메일** 버튼 클릭

3. **보내기** 클릭

[이미지: 회신 버튼 위치]

## 섹션 1: New Outlook 시작하기

### 1.1. 첫 로그인 & 계정 설정

> **처음 사용자**: 기본 로그인 방법은 [섹션 0: 5분 빠른 시작](#5분-빠른-시작-처음-사용자-필독)을 참조하세요.

- **회사 계정 형식**: `EP ID@poscointl1.onmicrosoft.com`
- **전환 순서 핵심**
  1) New Outlook 로그인으로 **새 메일 환경 활성화** →
  2) **EP 메일에서 자동 전달 규칙**을 설정(전환기간 병행 사용을 위한 임시조치)

#### 이메일 계정 추가 상세 단계
- **New Outlook 계정 추가 방법**
  1) 우측 상단 **설정(⚙️)** 아이콘 클릭
  2) **계정** > **이메일 계정** 선택
  3) **계정 추가** 버튼 클릭
  4) 추가할 이메일 주소 입력 (Outlook.com, Microsoft 365, Gmail, Yahoo, IMAP/POP 등 지원)
  5) 계정 유형에 따라 인증 절차 진행
  6) 비밀번호 입력 및 인증 완료
- **다중 계정 관리**: 여러 계정 추가 후 좌측 탐색 창에서 계정별로 전환 가능

#### EP 메일에서 Outlook으로 자동 전달 설정 (필수!)

> **중요**: EP 메일에서 New Outlook으로 공식 전환('25.11.10 예정) 전까지, EP의 모든 메일을 Outlook으로 자동 전달하여 Outlook에서 사용해야 합니다.

**따라하기 - EP 자동 전달 규칙 설정:**

1. **EP 메일 환경설정 열기**
   - EP 메일 접속 (기존 메일 시스템)
   - 우측 상단 **톱니바퀴 모양(⚙️)** 아이콘 클릭

[이미지: EP 메일 환경설정 버튼]

2. **규칙설정 메뉴 접근**
   - **규칙설정** 클릭

3. **새로운 규칙 설정**
   - **새로운 규칙 설정** 버튼 클릭

4. **타인에게 자동전달 선택**
   - 메일 규칙 목록에서 **타인에게 자동전달** 클릭

[이미지: EP 규칙설정 화면]

5. **전달받는 사람 입력**
   - "전달받는 사람"  필드에 `[본인 EP ID]@poscointl1.onmicrosoft.com` 입력
   - 예시: EP ID가 `khkang120`이면 → `khkang120@poscointl1.onmicrosoft.com`

6. **저장**
   - **추가** 버튼 클릭
   - **확인** 버튼 클릭

✅ **설정 완료!** 이제 EP로 오는 모든 메일이 자동으로 Outlook으로 전달됩니다.

**자동 전달 확인 방법:**
- Outlook에서 받은 메일 제목을 확인하세요
- 제목 앞에 `[Auto Fwd : 본인성명<이메일 주소@poscointl.com>]`가 표시됩니다
- 이 표시가 있으면 EP에서 자동 전달된 메일입니다

[이미지: Auto Fwd 표시 예시]

> **TIP** 전환 초기에는 "기본 메일함"을 Outlook로 쓰되, EP에서 온 메일이 누락되지 않도록 **자동 전달**을 반드시 켭니다. (자세한 지침은 [전환기 체크리스트](#전환기에poutlook-체크리스트--faq) 참조)

### 1.2. 인터페이스 둘러보기
- **탐색 창(Navigation Pane)**: 메일(CTRL+1), 캘린더(CTRL+2), 연락처/People(CTRL+4), To Do
- **리본(Ribbon)**: 현재 보기(메일/캘린더 등)별 명령
- **폴더 창**: 받은편지함/보낸편지함/지운편지함 등
- **메시지 목록 & 읽기 창**: 중앙 목록 + 우측 미리보기

> **빠른 적응 포인트**: 보기를 **대화형(Conversation)**으로 전환, 읽기 창 배치, 열(From/Subject/Received) 정렬을 본인 업무흐름에 맞게 커스터마이즈.

### 1.3. 필수 초기 환경 설정

#### 시간대 및 언어 설정
- **시간대 변경 방법**
  1) 우측 상단 **설정(⚙️)** 아이콘 클릭
  2) **일반** > **언어 및 시간** 선택
  3) **시간대** 드롭다운에서 현재 위치 선택
     - 예: `(UTC+09:00) 서울`, `(UTC-05:00) 뉴욕`, `(UTC+00:00) 런던`
  4) 변경 사항은 자동 저장됨
- **언어 설정**
  - 같은 메뉴에서 **표시 언어** 선택
  - 한국어, 영어 등 선택 가능
  - 브라우저 재시작 시 적용
- **캘린더 시간대 옵션**
  - **여러 시간대 표시**: 캘린더 보기 > 설정 > **시간대** 에서 추가 시간대 표시 가능
  - **출장 시 활용**: 본사(서울)와 현지 시간대를 동시에 표시하여 회의 일정 조율

#### 데스크톱 알림 설정
- **경로**: 설정(⚙️) > **일반** > **알림**
- **옵션**
  - 새 메일 도착 시 알림 팝업
  - 알림 소리 켜기/끄기
  - 미리보기 내용 표시 수준 (제목만/발신자+제목/전체 미리보기)
  - VIP 발신자별 특별 알림 설정

#### 테마 및 보기 설정
- **테마 변경**: 설정(⚙️) > **일반** > **모양**
  - **밝게**: 기본 밝은 테마
  - **어둡게**: 다크 모드 (야간 작업, 눈의 피로 감소)
  - **시스템 설정 사용**: Windows/Mac 시스템 설정 따라가기
- **메시지 밀도**: 컴팩트/기본/넓게 중 선택
- **대화형 보기**: 같은 주제의 메일을 스레드로 묶어 표시
- **읽기 창 위치**: 우측(기본)/하단/숨기기

## 섹션 2: 이메일 관리 마스터하기

### 2.1. 작성·회신·전달 기본기
- **기본 작업**: 새 메일 작성, 회신, 전체회신, 전달, 보내기
- **본문 편집**: 굵게, 기울임, 밑줄, 하이퍼링크 삽입
- **추가 기능**: @멘션으로 주의 환기, 이모지 삽입

> **단축키**: 전체 단축키 목록은 [섹션 8.1 핵심 단축키](#빠른-참조치트시트리소스)를 참조하세요.

### 2.2. 첨부 방식 & 보안 지침
> **핵심 개념**: **파일 '본문 첨부'** vs **OneDrive/SharePoint '링크 공유'**
> 사외/그룹사 수신 시, **링크**는 암호화 정책으로 **열람 불가**가 될 수 있어 **본문 첨부 권장**.

- **권장 사항(그룹사·사외 발송)**
  - **파일 본문 첨부**로 발송
  - 그룹사 발송 시 **"복호화 발송"** 옵션 사용(정책 오픈 일정에 따라 메뉴 제공)
- **해외 주재원(@poscointl.com) 발송 유의**
  - 정식 오픈 전 **복호화 필요** 시
    1) 문서 **보안 레이블을 '일반'**으로 변경해 발송 **또는**
    2) **EP 메일**을 이용해 발송
- **드래그&드롭 첨부 UX**
  - **좌측 영역으로 드롭**: 링크 첨부 (OneDrive에 업로드되어 링크로 첨부)
  - **우측 영역으로 드롭**: 본문 첨부 (파일이 직접 첨부됨)
  - **삽입 메뉴 사용**: 메뉴 > **삽입** > **파일 첨부** 에서도 "링크 공유" 또는 "파일 첨부" 선택 가능

[이미지: 드래그 앤 드롭 시 좌측 링크 첨부, 우측 본문 첨부 구분]

**첨부 방식 구분 방법:**
- 링크 첨부: 메일 본문에 파일 아이콘과 함께 "OneDrive에 업로드됨" 표시
- 본문 첨부: 메일 하단 "첨부 파일" 영역에 파일 표시

> **보안 실수 방지 체크**
> - 외부/그룹사: **본문 첨부 + (필요 시) 복호화 발송**
> - 내부(사내): 보안 레이블 정책 준수, 필요시 링크 공유

### 2.3. 고급 검색 기술
- 검색 상자 클릭 → 제안 검색어 확인
- **연산자 예시**
  - `from:홍길동` / `subject:정산` / `hasattachment:yes` / `received:today`
  - `"정확한 문구"`(따옴표)
- 리본의 **검색 필터**(보낸사람, 제목, 날짜 범위, 첨부 등)로 결과 협폭

### 2.4. 받은편지함 정리 자동화
- **폴더 전략**: 프로젝트/업무유형별 생성
- **빠른 정리**: **Archive(보관)**, **Sweep(정리)**
- **규칙**
  - **메시지에서 바로**: 우클릭 → **규칙 만들기**
  - **고급 규칙**: 다중 조건/다중 작업(특정 발신자+제목 키워드+나 참조 → 이동+읽음표시+자동 전달 등)
  - **우선순위 제어**: "더 이상 규칙 처리 안 함" 옵션으로 체인 중단

#### 📝 실습 예제: 규칙 만들기 따라하기

**시나리오 1: 특정 발신자의 메일을 자동으로 폴더로 이동**

*상황*: 프로젝트 A 담당자(김철수)가 보내는 메일이 많아서 별도 폴더에서 관리하고 싶습니다.

**따라하기:**
1. **폴더 만들기**
   - 좌측 폴더 목록에서 우클릭 > **새 폴더**
   - 폴더 이름: "프로젝트 A"

2. **규칙 만들기**
   - 김철수가 보낸 메일 하나를 선택
   - 메일 우클릭 > **규칙** > **규칙 만들기**
   - 조건: **보낸 사람이 김철수**
   - 작업: **폴더로 이동** > "프로젝트 A" 폴더 선택
   - **저장** 클릭

3. **기존 메일에도 적용할지 선택**
   - "받은편지함의 기존 메일에도 적용" 체크
   - **확인**

✅ **결과**: 이제 김철수가 보내는 모든 메일이 자동으로 "프로젝트 A" 폴더로 이동합니다!

---

**시나리오 2: 제목에 특정 키워드가 있는 메일을 빨간색으로 표시**

*상황*: 제목에 "[긴급]"이 포함된 메일을 눈에 띄게 하고 싶습니다.

**따라하기:**
1. **설정** > **메일** > **규칙** 이동
2. **새 규칙** 클릭
3. 규칙 이름: "긴급 메일 표시"
4. **조건 추가**:
   - "제목에 특정 단어 포함"
   - 키워드: `[긴급]` 또는 `긴급` 입력
5. **작업 추가**:
   - "카테고리로 표시" > **빨간색** 선택
6. **저장**

✅ **결과**: 제목에 "긴급"이 들어간 메일은 빨간색으로 강조 표시됩니다!

---

**시나리오 3: 뉴스레터 자동 정리**

*상황*: `newsletter@company.com`에서 오는 뉴스레터를 읽은 후 자동으로 보관하고 싶습니다.

**따라하기:**
1. 뉴스레터 메일 선택
2. 우클릭 > **규칙** > **규칙 만들기**
3. 조건: **보낸 사람이 newsletter@company.com**
4. 작업:
   - **읽음으로 표시** (선택사항)
   - **폴더로 이동** > "보관" 폴더
5. **저장**

📌 **Tip**: "더 이상 규칙 처리 안 함" 옵션을 켜면 이 규칙 이후의 다른 규칙이 적용되지 않습니다.

---

**시나리오 4: 내가 참조(CC)에만 있는 메일은 별도 폴더로**

*상황*: 내가 주 수신자가 아닌 참조 메일만 따로 보고 싶습니다.

**따라하기:**
1. **설정** > **규칙** > **새 규칙**
2. 조건:
   - "참조(CC) 라인에 내 이름이 있음"
   - "받는 사람 라인에 내 이름이 **없음**" (조건 추가)
3. 작업:
   - "폴더로 이동" > "참조 메일" 폴더
   - "카테고리" > 회색 (선택사항)
4. **저장**

✅ **결과**: 참조로만 받은 메일이 자동으로 분류되어 중요한 메일에 집중할 수 있습니다!

**규칙 관리 팁:**
- 규칙이 너무 많으면 순서가 중요! 위에서 아래로 실행됩니다.
- 규칙 테스트: 새 규칙 만든 후 테스트 메일 보내서 확인
- 작동하지 않으면: 설정 > 규칙에서 **켜짐/꺼짐** 상태 확인

#### 카테고리, 플래그, 미리 알림 활용
- **카테고리(Categories)**
  - 메일을 색상별로 분류 (예: 빨강=긴급, 파랑=프로젝트A, 초록=검토필요)
  - **설정**: 메일 선택 > 홈 탭 > **카테고리** > 색상 선택
  - **사용자 지정**: 카테고리명 변경, 단축키 할당 가능
  - **다중 카테고리**: 하나의 메일에 여러 색상 적용 가능
- **플래그(Flags)**
  - 후속 조치가 필요한 메일 표시
  - **빠른 플래그**: 메일 우클릭 > **플래그 설정**
  - **기한 설정**: 오늘, 내일, 이번 주, 다음 주, 사용자 지정 등
  - **플래그된 메일 보기**: To Do 목록에 자동 표시
- **미리 알림(Reminders)**
  - 특정 시간에 메일 알림 받기
  - **설정**: 메일 선택 > 홈 탭 > **후속 작업** > **미리 알림 추가**
  - 지정된 날짜/시간에 팝업 알림 표시
- **자동 부재중 회신**
  - **설정 경로**: 설정(⚙️) > **메일** > **자동 회신**
  - **옵션**
    - 시작/종료 날짜 및 시간 지정
    - 조직 내부용 메시지 (사내 직원용)
    - 조직 외부용 메시지 (외부 발신자용)
    - 외부 회신 범위: 모든 외부 발신자 또는 연락처만
  - **활용 시나리오**: 휴가, 출장, 회의 집중 기간 등

### 2.5. 전문 서명 만들기

**POSCO 임직원**: 회사 제공 표준 서명 템플릿 사용 방법은 [섹션 6.3 서명 템플릿 앱](#서명-템플릿-앱---회사-서명-관리)을 참조하세요.

**일반 서명 생성** (개인 용도):
- **접근**: 설정(⚙️) > **메일** > **작성 및 회신** > **서명**
- **구성**: 이름·직함·부서·회사·연락처·로고(이미지 크기 조절)
- **자동삽입 설정**: 새메시지 / 회신·전달 각각 지정 가능

## 섹션 3: 캘린더 & 일정 고급 활용

### 3.1. 약속·회의 예약
- **약속**(개인 일정): **CTRL+SHIFT+A**
- **모임**(참석자 초대): **CTRL+SHIFT+Q**
- **일정 관리 도우미**(Scheduling Assistant): 참석자 가용시간 겹침 최소화
- **회의실 찾기**(Room Finder): 사용 가능 회의실 즉시 예약
- **되풀이**: 주간/월간 반복 설정

### 3.2. Microsoft Teams 연동
- 일정 창에서 **Teams 모임** 토글만 켜면 본문에 **참가 링크** 자동 삽입

### 3.3. 공유·권한·대리인
- **공유 시작**: 캘린더 보기 > **홈 탭 > 캘린더 공유**
- **권한 수준**
  - **바쁨/한가함만**
  - **제목+위치 보기**
  - **모든 세부 정보 보기**
  - **편집 가능**
  - **대리인(Delegate)**: 모임 요청/응답을 대신 처리
- **대리인 옵션**: 개인 항목 표시 허용, 응답 수신처(본인/대리인/둘 다) 지정
- **공유 수락**: 동료 초대 수락 → 내 캘린더 목록에 추가

> **전환기 주의**: Classic Outlook은 **파일 > 계정 설정 > 대리인 액세스** 경로, New Outlook은 **캘린더 공유** 화면 내 권한 설정에 통합(자세한 비교는 부록 참조).

## 섹션 4: 연락처(피플) 관리

### 4.1. 개별 연락처
- **이동**: 탐색창 **People**(CTRL+4)
- **생성**: '새 연락처' 또는 메일 보낸사람 **프로필 카드 → 연락처 추가**
- **편집**: 선택 후 세부정보 창에서 수정

### 4.2. 연락처 목록(그룹)
- **명칭 차이**: Classic **연락처 그룹(Contact Group)** ↔ New **연락처 목록(Contact List)**(동일 기능)
- **생성**: People > 홈 > **새 연락처 목록**
- **구성원**: 주소록 검색 추가 또는 이메일 직접 입력

> **POSCO 전용**: 그룹사 조직도 및 배포그룹 검색은 [섹션 6.1 수신처 앱](#수신처-앱---조직도-및-배포그룹-활용)을 사용하세요.

### 4.3. CSV 가져오기/내보내기
- **내보내기**(Classic): 파일 > 열기 및 내보내기 > 가져오기/내보내기 → CSV
- **가져오기**(New): People > **연락처 관리 > 연락처 가져오기** → CSV 업로드
- **중복 처리 옵션**
  - 가져온 항목으로 **바꾸기** / **중복 허용** / **가져오지 않음**
- **CSV 준비**: 첫 행에 '이름, 성, 이메일 주소…' 등 표준 열제목

## 섹션 5: Microsoft To Do 통합

### 5.1. To Do 소개
- Outlook 내장 **할 일** 앱
- **스마트 목록**: **My Day**, **중요**, **계획된 일정** 등

- **드래그 & 드롭**: 받은 메일을 **My Day**로 끌어 놓으면 **제목·원본 메일 링크 포함 작업** 자동 생성
- **효과**: Inbox Zero(보관/삭제), 누락 방지, 메일은 유입 통로·실행은 To Do에서 관리

## 섹션 6: POSCO 커스텀 앱 완벽 가이드

> **중요**: 이 섹션은 POSCO INTERNATIONAL 전용 커스텀 앱 5가지를 다룹니다. 일반 Outlook에는 없는 기능입니다.

### 📌 커스텀 앱 개요

POSCO INTERNATIONAL에서 업무 효율성을 위해 추가로 개발한 **5가지 전용 기능**입니다. EP의 핵심 기능을 Outlook에서도 사용할 수 있도록 구현했습니다.

**중요 제약사항**:
- **모바일 Outlook 앱에서는 커스텀 기능을 사용할 수 없습니다**
- PC 앱 또는 웹 버전(`https://outlook.office.com`)을 사용하세요

**5가지 커스텀 앱 목록:**
1. **수신처 앱**: 조직도 및 배포그룹 조회
2. **대용량 첨부 앱**: 최대 500MB 파일 전송
3. **서명 템플릿 앱**: 회사 표준 서명 관리
4. **수신확인 앱**: 메일 읽음 확인 및 회수
5. **EP 스마트회의 일정 연동**: 회의실 예약 자동 연동

---

### 6.1. 수신처 앱 - 조직도 및 배포그룹 활용

#### 📖 개요

EP 메일 작성 시 사용하던 **'배포그룹'** 기능과 동일한 기능으로, **P-인터내셔널 및 그룹사 조직도**에서 수신인을 검색하여 메일을 발송할 수 있습니다.

**주요 기능**:
- 부서별 조직도 탐색
- 배포그룹(메일링 리스트) 선택
- 그룹사 직원 검색

---

#### 🎬 사용 방법

**Step 1: 새 메일 작성 시작**

1. Outlook 상단 탭에서 **[홈]** 클릭
2. **[새 메일]** 버튼 클릭
3. 메일 작성 창이 열림

**Step 2: 수신처 앱 실행**

1. 메일 작성 창 상단 리본 메뉴에서 **[수신처]** 앱 아이콘 클릭
2. 우측에 수신처 패널이 열림

> 💡 **Tip**: 리본 메뉴가 축소된 경우 **[...]** 더보기 버튼을 눌러 수신처 앱을 찾으세요.

**Step 3: 조직도에서 수신인 선택**

**방법 A: 조직도 탐색**
1. 수신처 패널 상단에서 **[조직도]** 탭 선택
2. 조직 구조를 펼쳐서 원하는 부서 찾기
   - 예: `IT혁신실` → `디지털혁신팀`
3. 직원 이름 옆 체크박스 선택
4. 하단 **[받는 사람]** 또는 **[참조]** 버튼 클릭

**방법 B: 배포그룹 선택**
1. 수신처 패널 상단에서 **[배포그룹]** 탭 선택
2. 그룹명 검색 (예: "전사공지", "부서장")
3. 그룹 선택 후 **[받는 사람]** 클릭

**방법 C: 검색 기능**
1. 수신처 패널 상단 검색창에 이름 입력
2. 검색 결과에서 선택
3. **[받는 사람]** 버튼 클릭

**Step 4: 수신인 등록 확인**

1. 메일 작성 창의 **받는 사람** 필드에 선택한 수신인이 자동 입력됨
2. 여러 명 선택 시 쉼표(,)로 구분되어 표시
3. 제목과 본문 작성 후 **[보내기]** 클릭

---

#### ⚠️ 주의사항

- **외부 이메일**: 외부 업체나 고객의 이메일 주소는 수신처 앱에서 검색되지 않으므로 직접 입력해야 합니다.

#### 💡 활용 팁

**시나리오 2: 타 부서 담당자 찾기**
- 조직도에서 해당 부서를 펼쳐 담당자 확인 후 선택

**시나리오 3: 여러 부서 담당자에게 동시 발송**
- 조직도에서 여러 부서를 탐색하며 담당자를 하나씩 체크박스로 선택

### 6.2. 대용량 첨부 앱 - 500MB 파일 전송

#### 📖 개요

일반 Outlook의 첨부 용량 제한(약 30MB)을 극복하기 위한 앱으로, **EP와 동일하게 최대 500MB, 20개 파일**까지 첨부할 수 있습니다.

**주요 기능**:
- 대용량 파일 첨부 (합계 500MB, 최대 20개)
- Drag & Drop 업로드
- 첨부 완료 후 다운로드 링크 생성

---

#### 🎬 사용 방법

**Step 1: 대용량 첨부 앱 실행**

1. 메일 작성 창 상단 리본 메뉴에서 **[대용량 첨부]** 앱 아이콘 클릭
2. 대용량 첨부 창이 팝업으로 열림

**Step 2: 파일 업로드**

**방법 A: Drag & Drop**
1. 업로드 영역(점선 박스)으로 파일을 드래그
2. 파일이 자동으로 업로드되며 진행률 표시
3. 여러 파일을 동시에 드래그 가능

**방법 B: 파일 선택**
1. **[파일 선택]** 버튼 클릭
2. 탐색기에서 파일 선택 (Ctrl 키로 여러 파일 선택 가능)
3. **[열기]** 버튼으로 업로드 시작

**Step 3: 첨부 완료**

1. 모든 파일 업로드 완료 후 **[첨부완료]** 버튼 클릭
2. 메일 본문에 다운로드 링크가 자동 삽입됨
3. 수신자는 링크를 클릭하여 파일 다운로드

> 📌 **중요**: 업로드 중에는 메일 작성 창을 닫지 마세요. 업로드가 중단될 수 있습니다.

---

#### 🌐 대용량 파일 유통 가이드 (수신자 기준)

| 수신자 위치 | 국내/해외 | 권장 방법 | 이유 |
|---|---|---|---|
| **P-인터내셔널** | 국내 | OneDrive/SharePoint 링크 | 속도 우수, 실시간 협업 가능 |
| **P-인터내셔널** | 해외 | OneDrive/SharePoint 링크<br>(민감도 레이블 '일반' 필수) | 해외 접근 가능, 보안 레이블 조정 필요 |
| **그룹사/사외** | 국내 | 대용량 첨부 앱 | 속도 이슈 없음 |
| **그룹사/사외** | 해외 | OneDrive/SharePoint 링크 권장 | 대용량 첨부 앱 사용 가능하나 속도 느림 |

> ⚠️ **해외 발송 시 주의**: OneDrive/SharePoint 링크 사용 시 수신자가 MS 계정을 보유하고 있어야 하며, 이메일 인증이 필요할 수 있습니다.

---

#### ⚠️ 주의사항

- **파일 형식 제한**: 실행 파일(.exe), 스크립트 파일(.bat, .sh) 등 보안상 위험한 파일은 업로드가 차단될 수 있습니다.
- **유효 기간**: 대용량 첨부 파일의 다운로드 링크는 **30일간 유효**합니다. 이후에는 다운로드할 수 없으므로 중요한 파일은 별도 백업하세요.
- **삭제 불가**: 첨부 완료 후에는 파일을 삭제하거나 수정할 수 없습니다. 재전송이 필요합니다.

#### 💡 활용 팁

**시나리오 1: 대용량 보고서 발송**
- 200MB PPT 파일을 대용량 첨부 앱으로 업로드하여 임원진에게 발송

**시나리오 2: 그룹사에 설계 도면 전송**
- 여러 CAD 파일(총 400MB)을 대용량 첨부 앱으로 전송

**시나리오 3: 해외 법인에 자료 공유**
- OneDrive에 업로드 후 링크 공유 방식 사용 (속도 및 접근성 우수)

### 6.3. 서명 템플릿 앱 - 회사 서명 관리

#### 📖 개요

회사에서 제공하는 **표준 이메일 서명 템플릿**을 클립보드로 복사하여 Outlook 서명으로 등록할 수 있는 앱입니다.

**주요 기능**:
- 회사 표준 서명 템플릿 제공 (동일 템플릿)
- 클립보드 복사 기능
- Outlook 서명 메뉴에 등록하여 자동 삽입

---

#### 🎬 사용 방법

**Step 1: 서명 템플릿 앱 실행**

1. 메일 작성 창 상단 리본 메뉴에서 **[서명 템플릿]** 앱 아이콘 클릭
2. 서명 템플릿 목록이 우측 패널에 표시됨

**Step 2: 서명 템플릿 복사**

1. 제공되는 서명 템플릿 선택
   - 예: "서명_2507278"
2. **[클립보드로 복사]** 버튼 클릭
3. "복사되었습니다" 메시지 확인

> 💡 **Tip**: 복사 후에는 반드시 Outlook 서명 관리 화면에서 등록해야 합니다.

---

#### 🎬 서명 등록 방법 (필수)

서명 템플릿을 복사한 후, **반드시 Outlook 서명 관리 화면에서 등록**해야 합니다.

> ⚠️ **중요**: 메일 본문에 직접 붙여넣기는 권장하지 않습니다. 아래 방법을 따라 Outlook 기본 서명으로 등록하세요.

**Step 1: Outlook 설정 열기**
1. Outlook 우측 상단 **[⚙️ 설정]** 아이콘 클릭
2. **[모든 Outlook 설정 보기]** 클릭

**Step 2: 서명 메뉴 진입**
1. 설정 창 좌측에서 **[메일]** 선택
2. **[작성 및 회신]** 탭 선택
3. 아래로 스크롤하여 **[서명]** 섹션 찾기

**Step 3: 새 서명 생성**
1. **[+ 새 서명]** 버튼 클릭
2. 서명 이름 입력 (예: "서명_2025")
3. 서명 편집기 영역에 **Ctrl + V**로 복사한 서명 붙여넣기
4. 개인 정보 수정:
   - 이름, 직급, 부서명 확인 및 수정
   - 전화번호, 이메일 주소 확인
   - 모바일 번호 추가 (선택 사항)

**Step 4: 자동 삽입 설정**
1. 서명 편집 영역 하단에서 다음 옵션 설정:
   - **"새 메시지의 경우"**: 방금 만든 서명 선택
   - **"회신/전달의 경우"**: 서명 선택 또는 "없음" (선택 사항)
2. **[저장]** 버튼 클릭
3. 설정 창 닫기

**Step 5: 서명 테스트**
1. 새 메일 작성
2. 메일 본문 하단에 서명이 자동 삽입되었는지 확인
3. 필요 시 서명 위치 조정

---

#### ⚠️ 주의사항

**EP vs Outlook 서명 기능 변경사항**

| 항목 | EP | Outlook |
|---|---|---|
| 서명 적용 방식 | 환경설정 메뉴에서 유형 선택 시 자동 적용 | 템플릿 복사 → 서명 메뉴에 수동 등록 후 사용 |
| 자동 변경 | 회사 정보 변경 시 자동 업데이트 | 수동 수정 필요 (직급, 부서 변경 시) |

> 📌 **중요**: 직급이나 부서가 변경된 경우, Outlook 서명을 직접 수정해야 합니다.

---

#### 💡 활용 팁

**시나리오: 회신/전달 시 서명 설정**
- 새 메일 작성 시: 자동으로 전체 서명 삽입
- 회신/전달 시: 서명을 삽입하지 않거나 간소화된 서명 사용
- Outlook 서명 설정에서 "회신/전달의 경우" 옵션으로 조정 가능

### 6.4. 수신확인 앱 - 메일 확인 및 회수

#### 📖 개요

발송한 메일에 대해 **수신 확인 및 메일 회수** 기능을 제공하는 앱입니다.

**주요 기능**:
- 발송 메일별 수신자의 읽음 여부 확인
- 읽은 메일도 회수 가능
- 수신자별 읽음 상태 표시

**제약 사항**:
- **@poscointl.com 계정 사용자에게만 적용**
- 그룹사 및 사외로 발송한 메일은 수신확인 및 회수 불가

---

#### 🎬 사용 방법

**Step 1: 보낸 편지함에서 메일 선택**

1. Outlook 좌측 폴더 목록에서 **[보낸 편지함]** 클릭
2. 수신 확인하고 싶은 메일 선택

**Step 2: 수신확인 앱 실행**

1. 메일 선택 상태에서 상단 리본 메뉴의 **[수신확인]** 앱 아이콘 클릭
2. 우측에 수신확인 패널이 열림

**Step 3: 수신 여부 확인**

수신확인 패널에서 각 수신자의 상태가 아이콘으로 표시됩니다:

| 상태 아이콘 | 의미 |
|---|---|
| ✅ **메일 읽음** | 수신자가 메일을 열어봄 |
| ⬜ **읽지 않음** | 수신자가 아직 메일을 읽지 않음 |
| 🌐 **외부 수신자** | 그룹사/사외 수신자 (확인 불가) |
| ❌ **메일 없음 - 회수됨** | 회수가 완료된 메일 |

> 💡 **Tip**: 수신자 이름을 클릭하면 상세 정보(수신 시각, 읽은 시각)를 확인할 수 있습니다.

---

#### 🎬 메일 회수 방법

**Step 1: 회수할 수신자 선택**

1. 수신확인 패널에서 **회수하고 싶은 수신자** 체크박스 선택
2. 여러 명을 동시에 선택 가능

> ⚠️ **주의**: 외부 수신자(그룹사/사외)는 선택할 수 없습니다.

**Step 2: 회수 실행**

1. 하단 **[회수]** 버튼 클릭
2. 확인 메시지 팝업에서 **[회수]** 다시 클릭
3. "회수가 완료되었습니다" 메시지 확인

**Step 3: 회수 결과 확인**

1. 수신확인 패널에서 회수된 수신자는 **"메일 없음 - 회수됨"** 상태로 변경됨
2. 수신자의 받은 편지함에서 해당 메일이 삭제됨

---

#### ⚠️ 주의사항

- **읽은 메일 회수**: 읽은 메일도 회수 가능합니다.
- **그룹사/사외 제약**: 그룹사 및 사외로 발송한 메일은 수신확인 및 회수가 불가능합니다. 발송 전에 내용을 반드시 확인하세요.
- **계정 제약**: @poscointl.com 계정 사용자에게만 적용됩니다.

---

#### 💡 활용 팁

**시나리오 1: 잘못 발송한 메일 즉시 회수**
- 첨부 파일을 잘못 첨부한 경우, 보낸 편지함에서 해당 메일을 선택하여 즉시 회수 후 재발송

**시나리오 2: 중요 메일의 읽음 여부 추적**
- 보고서 제출 후, 임원진이 메일을 읽었는지 수신확인 앱으로 확인

**시나리오 3: 읽은 메일 회수**
- 보안상 문제가 있는 내용이 포함된 메일을 수신자가 이미 읽었더라도 회수 가능

### 6.5. EP 스마트회의 일정 연동

#### 📖 개요

EP의 **스마트회의(회의실 예약 시스템)**에서 회의실을 예약하면, **Outlook/Teams 일정**에 자동으로 등록되는 기능입니다.

**주요 기능**:
- EP 스마트회의에서 예약한 일정이 Outlook/Teams에 자동 등록
- 회의 일정 자동 동기화

**제약 사항**:
- **EP → Outlook 단방향만 지원** (Outlook → EP 역방향 연동 불가)

---

#### 🎬 사용 방법

**EP 스마트회의에서 회의실 예약**

1. EP 포털 접속
2. **[스마트회의]** 앱 실행
3. 회의실 예약 정보 입력 및 예약 완료

**Outlook/Teams 일정 확인**

EP 스마트회의에서 예약한 회의가 자동으로 Outlook 및 Teams 일정에 등록됩니다.

1. **Outlook**: 좌측 메뉴에서 **[일정]** 클릭 후 예약한 날짜 확인
2. **Teams**: 좌측 메뉴에서 **[일정]** 클릭 후 회의 확인

---

#### ⚠️ 주의사항

- **단방향 연동**: EP → Outlook/Teams로만 연동됩니다. Outlook에서 일정을 수정하거나 삭제해도 EP 스마트회의에는 반영되지 않습니다.
- **회의 변경**: 회의 일정 변경이 필요한 경우 **반드시 EP 스마트회의에서 수정**하세요.

Outlook 모바일 앱은 이동 중에도 생산성을 유지할 수 있도록 설계되었습니다.

- **최근 검색**: 자주 찾는 내용 빠르게 다시 검색

- **빠른 연락처 추가**: 메일에서 발신자 탭 > **연락처에 추가**

- **알림 사용자 지정**: VIP별, 프로젝트별 알림 설정

## 섹션 9: 전환기(EP→Outlook) 체크리스트 & FAQ

### 9.1. 전환 체크리스트(요약)
- [ ] **Outlook 로그인**(회사 계정)

- [ ] **첨부 정책 준수**: 그룹사/사외는 **본문 첨부 + (필요 시) 복호화 발송**
- [ ] **서명/규칙/보기** 개인화
- [ ] **모바일 Outlook** 설치(필요 시) & 알림 설정

### 9.2. 자주 묻는 질문(FAQ)

**Q1. 수신확인/회수는 되나요?**
A. **가능**합니다. 수신확인 앱(섹션 6.4)을 사용하세요. 단, **@poscointl.com 사용자에게만** 가능하며, 그룹사/사외는 불가합니다. 읽은 메일도 회수 가능합니다.

**Q2. 그룹사에 문서가 안 열린다는데요?**
A. 링크 첨부는 그룹사에서 열람 불가할 수 있습니다. 자세한 첨부 방식 및 보안 지침은 [섹션 2.2](#첨부-방식--보안-지침)를 참조하세요.

**Q3. 해외 주재원(@poscointl.com)에게 복호화 문서를 보내려면?**
A. [섹션 2.2](#첨부-방식--보안-지침)의 "해외 주재원 발송 유의" 항목을 참조하세요.

**Q4. EP→Outlook 자동 전달이 켜졌는지 어떻게 알아요?**
A. Outlook 받은 메일 제목에 **`[Auto Fwd : 본인성명<이메일>]`** 표기가 보입니다.

**Q5. 그룹사 주소가 주소록에 없어요.**
A. **수신처 앱**(섹션 6.1)을 사용하세요. 배포그룹 기능으로 그룹사 조직정보를 조회할 수 있습니다.

**Q6. 커스텀 앱이 보이지 않아요.**
A. [섹션 11.6 커스텀 앱 문제](#커스텀-앱-문제) 해결 가이드를 참조하세요.

**Q7. 대용량 첨부 앱으로 보낸 파일을 수신자가 다운로드할 수 없다고 합니다.**
A.
- **링크 유효 기간**: 30일 후 자동 삭제되므로 유효 기간 확인
- **IT 헬프데스크 문의**: 다운로드 문제 발생 시 IT 헬프데스크에 문의하세요.

**Q8. 대용량 첨부 앱과 OneDrive 링크 첨부의 차이는?**
A. 대용량 첨부 앱은 500MB/20개 파일까지 지원하며, OneDrive는 용량 제한이 없습니다. 자세한 비교표는 [섹션 6.2 대용량 첨부 앱](#대용량-첨부-앱---500mb-파일-전송)의 "대용량 파일 유통 가이드" 표를 참조하세요.

**Q9. 서명을 여러 개 등록할 수 있나요?**
A. Outlook 서명 메뉴에서 여러 개 등록 후, 메일 작성 시 필요에 따라 선택하여 사용할 수 있습니다.

**Q10. EP 스마트회의에서 회의를 예약했는데, Outlook 일정에 나타나지 않아요.**
A. EP 스마트회의에서 예약한 회의는 자동으로 Outlook/Teams 일정에 등록됩니다. 연동에 시간이 소요될 수 있으므로 잠시 후 새로고침하여 확인하세요.

| 기능 | **New Outlook** | **Classic Outlook** |
|---|---|---|

## 섹션 11: 문제 해결 가이드

### 11.1. 로그인 문제

**증상: "계정을 찾을 수 없습니다" 오류**
- **원인**: 잘못된 계정 형식 입력
- **해결**:
  1. 계정 형식 확인: `[EP ID]@poscointl1.onmicrosoft.com`
  2. @뒤의 도메인이 정확한지 확인
  3. 대소문자 구분 없음

**증상: 비밀번호가 틀렸다고 나옴**
- **해결**:
  1. Caps Lock이 켜져있는지 확인
  2. 회사 비밀번호 변경 후 새 비밀번호로 시도
  3. IT 헬프데스크에 비밀번호 초기화 요청

**증상: 2단계 인증 문제**
- **해결**:
  1. 등록된 휴대폰으로 인증 코드 확인
  2. 인증 앱(Microsoft Authenticator) 사용
  3. 인증 수단을 잃어버린 경우 IT 헬프데스크 문의

---

### 11.2. 메일 발송 문제

**증상: "보내지 못했습니다" 오류**
- **가능한 원인과 해결**:
  1. **큰 첨부 파일**: 25MB 이하로 줄이거나 대용량 첨부 앱(섹션 6.2) 사용
  2. **잘못된 이메일 주소**: 받는 사람 주소 재확인
  3. **네트워크 연결 끊김**: 인터넷 연결 확인 후 재시도

**증상: 외부로 메일이 안 나감**
- **해결**:
  1. 보안 정책 확인 (외부 발송 제한 여부)
  2. 보안 레이블이 적절한지 확인
  3. IT 정책에 따라 승인 필요할 수 있음

**증상: 첨부 파일이 안 보인다고 함 (그룹사/외부)**
- **원인**: 링크 첨부 방식 사용으로 권한 없음
- **해결**: 섹션 2.2 참조, **본문 첨부** 방식 사용

[이미지: 본문 첨부 vs 링크 첨부 구분]

---

### 11.3. 메일 수신 문제

**증상: 메일이 안 들어옴**
- **체크리스트**:
  1. **스팸 폴더** 확인 (정크 메일함)
  2. **규칙/필터** 확인 - 자동으로 다른 폴더로 이동했는지
  3. **받은편지함 보기** 설정 - "읽지 않음만 표시" 필터 해제
  4. 새로고침 (F5 또는 동기화 버튼)

**증상: 검색해도 메일이 안 나옴**
- **해결**:
  1. 검색 범위 확인 (현재 폴더 vs 모든 폴더)
  2. 검색 필터 제거 (날짜, 발신자 등)
  3. 정확한 키워드 사용 (띄어쓰기, 따옴표 활용)

---

### 11.4. 캘린더 문제

**증상: 회의 초대를 보냈는데 상대방이 못 받음**
- **해결**:
  1. 받는 사람 이메일 주소 확인
  2. **보내기** 버튼 눌렀는지 확인 (저장만 하면 안 보내짐)
  3. 보낸편지함에서 전송 여부 확인

**증상: 다른 사람의 캘린더가 안 보임**
- **해결**:
  1. 공유 권한 확인 (상대방이 공유했는지)
  2. 캘린더 목록에서 체크박스 켜기
  3. 캘린더 새로고침

**증상: Teams 링크가 자동으로 안 들어감**
- **해결**:
  1. 일정 작성 화면에서 **Teams 모임** 토글 켜기
  2. Teams 통합 설정 확인 (설정 > 캘린더)
  3. 수동으로 Teams 링크 생성 후 붙여넣기

---

### 11.5. 첨부 파일 문제

**증상: 첨부 파일이 너무 커서 안 보내짐**
- **해결**:
  1. **대용량 첨부 앱** 사용 (섹션 6.2, 최대 500MB)
  2. OneDrive/SharePoint 업로드 후 링크 공유 (사내 전용)
  3. 파일을 여러 개로 나누어 발송

**증상: 첨부한 파일이 사라짐**
- **원인**: 임시 저장 중 세션 만료
- **예방**: 작성 중 수시로 저장 (CTRL+S)
- **복구**: 임시 보관함에서 확인

---

### 11.6. 커스텀 앱 문제

**증상: 커스텀 앱이 보이지 않음**
- **해결**:
  1. 리본 메뉴 **[...]** 더보기 버튼 클릭
  2. 웹 버전(`https://outlook.office.com`) 사용
  3. 브라우저 캐시 및 쿠키 삭제 후 재로그인
  4. IT 헬프데스크 문의

**증상: 수신처 앱에서 그룹사 직원이 검색 안 됨**
- **해결**:
  1. 수신처 앱의 배포그룹 기능으로 그룹사 조직정보 조회
  2. 검색 기능을 사용하여 그룹사 직원 이름 입력
  3. IT 헬프데스크 문의

**증상: 대용량 첨부 앱 업로드 중 오류**
- **해결**:
  1. 파일 형식 확인 (.exe, .bat 등은 차단됨)
  2. 총 용량 500MB, 20개 파일 제한 확인
  3. 네트워크 연결 안정성 확인

---

### 11.7. 성능 문제

**증상: Outlook이 느려요**
- **해결**:

  2. **캐시 삭제**: 브라우저 캐시 및 쿠키 삭제 (웹 사용 시)

  4. **다른 브라우저** 시도 (Edge, Chrome)

**증상: 첨부 파일 다운로드가 안 됨**
- **해결**:
  1. 팝업 차단 해제
  2. 브라우저 다운로드 폴더 확인
  3. 바이러스 백신 프로그램 확인

  3. 모바일 앱 로그아웃 후 다시 로그인
  4. 앱 업데이트 확인

## 섹션 12: 용어 사전

### A-C

**Archive (보관)**
- 받은편지함에서 메일을 제거하되 삭제하지 않고 보관 폴더로 이동하는 기능
- 받은편지함을 깔끔하게 유지하면서 나중에 찾을 수 있음

**Attachment (첨부 파일)**
- 메일과 함께 보내는 파일 (문서, 이미지, 압축 파일 등)
- 본문 첨부 vs 링크 첨부 방식이 있음

**Auto-reply (자동 회신)**
- 휴가, 출장 등으로 부재 시 자동으로 답장을 보내는 기능
- "부재중 메시지" 또는 "Out of Office"라고도 함

**Calendar (캘린더)**
- 일정, 회의, 약속을 관리하는 Outlook의 일정 관리 도구

**Category (카테고리)**
- 메일이나 일정을 색상별로 분류하는 기능
- 예: 빨강=긴급, 파랑=프로젝트A

**CC (Carbon Copy, 참조)**
- 메일의 주 수신자는 아니지만 내용을 공유하고 싶은 사람
- 받는 사람: 답변이 필요한 사람 / CC: 참고만 하면 되는 사람

**Conversation View (대화형 보기)**
- 같은 주제의 메일들을 스레드(실타래)로 묶어서 보여주는 기능
- 메일 체인을 따라가기 쉬움

---

### D-I

**Delegate (대리인)**
- 다른 사람이 본인의 메일이나 캘린더를 대신 관리할 수 있도록 권한을 주는 기능
- 비서가 상사의 일정을 관리할 때 사용

**Draft (임시 보관함)**
- 작성 중이지만 아직 보내지 않은 메일이 저장되는 곳
- 자동 저장되며 나중에 이어서 작성 가능

**Flag (플래그)**
- 후속 조치가 필요한 메일에 깃발 표시
- 플래그된 메일은 To Do 목록에 자동으로 추가됨

**Focused Inbox (중요 받은편지함)**
- AI가 중요한 메일과 덜 중요한 메일을 자동으로 분류
- 모바일 앱에서 주로 사용

**Forwarding (전달)**
- 받은 메일을 다른 사람에게 보내는 기능
- 원본 메일 내용이 포함됨

**IMAP/POP**
- 외부 이메일 계정(Gmail, Yahoo 등)을 Outlook에 연결하는 프로토콜
- IMAP: 서버와 동기화 / POP: 다운로드

**Inbox (받은편지함)**
- 새로 도착한 메일이 쌓이는 기본 폴더

**Inbox Zero**
- 받은편지함을 비우는 이메일 관리 방법론
- 모든 메일을 처리, 보관, 삭제하여 받은편지함을 0으로 유지

---

### J-R

**Junk Mail (정크 메일, 스팸)**
- 원하지 않는 광고, 피싱 메일 등
- 자동으로 정크 메일함으로 이동

**Meeting Request (모임 요청)**
- 다른 사람을 회의에 초대하는 일정 초대장
- 수락/거절/미정 응답 가능

**Navigation Pane (탐색 창)**
- Outlook 좌측의 메뉴 영역
- 메일, 캘린더, 연락처, To Do 탭이 있음

**Out of Office (부재 중)**
- Auto-reply와 동일한 의미
- 부재중 자동 응답 메시지

**People (피플, 연락처)**
- 이메일 주소, 전화번호 등 연락처 정보를 저장하는 곳
- 주소록

**PWA (Progressive Web App)**
- 웹 앱을 마치 설치된 앱처럼 사용할 수 있게 하는 기술
- Outlook Web을 PWA로 설치하면 오프라인에서도 제한적 사용 가능

**Reading Pane (읽기 창)**
- 메일 목록에서 선택한 메일의 내용을 보여주는 영역
- 우측 또는 하단에 배치 가능

**Recurring (되풀이, 반복)**
- 매주, 매월 등 정기적으로 반복되는 일정
- 예: 매주 월요일 10시 주간 회의

**Reminder (미리 알림)**
- 특정 시간에 팝업으로 알려주는 기능
- 메일, 일정, 작업에 모두 설정 가능

**Reply (회신)**
- 받은 메일에 답장하는 기능
- Reply All (전체 회신): 모든 수신자에게 답장

**Ribbon (리본)**
- Outlook 상단의 명령 도구 모음
- 파일, 홈, 보기 등 탭으로 구성

**Room Finder (회의실 찾기)**
- 사용 가능한 회의실을 자동으로 찾아주는 기능

**Rules (규칙)**
- 특정 조건의 메일을 자동으로 처리하는 자동화 기능
- 예: "김철수가 보낸 메일은 자동으로 프로젝트A 폴더로 이동"

---

### S-Z

**Scheduling Assistant (일정 관리 도우미)**
- 여러 사람의 일정을 겹쳐서 보여주는 기능
- 모두가 참석 가능한 시간을 찾기 쉬움

**Search (검색)**
- 메일, 일정, 연락처 등을 키워드로 찾는 기능
- 연산자 사용 가능 (from:, subject: 등)

**Signature (서명)**
- 메일 하단에 자동으로 삽입되는 발신자 정보
- 이름, 직함, 연락처, 회사 로고 등 포함

**Sweep (정리)**
- 특정 발신자의 메일을 한 번에 정리하는 기능
- 뉴스레터 등 대량 메일 삭제에 유용

**Sync (동기화)**
- 여러 기기(PC, 모바일, 웹) 간 데이터를 일치시키는 것
- 한 곳에서 읽은 메일이 다른 곳에서도 읽음 표시됨

**Teams Meeting (Teams 모임)**
- Microsoft Teams 화상 회의 링크가 포함된 일정
- Outlook에서 직접 생성 가능

**Thread (스레드)**
- 같은 주제로 주고받은 메일들의 연결된 흐름
- 대화형 보기에서 하나로 묶임

**To Do (할 일)**
- Outlook과 통합된 작업 관리 앱
- 플래그된 메일이 자동으로 To Do에 표시됨

**vCard (.vcf)**
- 연락처 정보를 담은 파일 형식
- 연락처를 파일로 주고받을 때 사용

**VIP**
- 중요한 발신자를 지정하는 기능
- VIP의 메일은 특별 알림 설정 가능

**Web Version (웹 버전)**
- 브라우저에서 사용하는 Outlook (outlook.office.com)
- 설치 없이 어디서나 접근 가능
