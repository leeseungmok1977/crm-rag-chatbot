## 섹션 0: 5분 빠른 시작 (처음 사용자 필독)

### ✅ 체크리스트
- [ ] 회사 계정으로 CRM 접속 (M365/SSO 기준)
- [ ] 내 부서·거래선·담당 프로젝트 북마크 저장
- [ ] 거래선·연락처 기본 조회/검색 방법 익히기
- [ ] '나의 오픈 기회/주문' 즐겨찾기
- [ ] 미팅 메모 템플릿으로 첫 메모 작성 & 할당

### 첫 접속
1) Edge/Chrome 실행 → 회사 SSO → CRM URL 접속
2) 언어(한국어/English), 시간대(Seoul) 확인
3) 홈(대시보드)에서 **나의 파이프라인/주요 알림** 확인

### 가장 많이 쓰는 3가지
1) **거래선(Account) 찾기**: 이름/사업자번호/국가로 검색 → '별표'로 즐겨찾기
2) **연락처(Contact) 추가**: 거래선 상세 > 연락처 탭 > '새 연락처'
3) **미팅 메모 작성**: 미팅 일정/참석자 선택 → 주요 안건·Action Item 기록 → 담당자·기한 지정


## 섹션 1: CRM 개요 및 접근

### CRM 범위
- **거래선·연락처** (Account/Contact) 마스터 관리
- **주문/수주 → 이행(출고/선적/정산)** 전 과정 추적
- **미팅 메모** 기록·공유 및 후속조치(Action Item) 관리
- 공통 코드(Master) 기반의 표준화/통계 및 리포트

### 접근 경로 & 권장 브라우저
- Microsoft 365 계정(SSO) / Edge 또는 Chrome 최신 버전
- 다국어 UI 지원(한국어/영문), 사용자 단위 언어 선택

### 검색
- 전역 검색: 이름/코드/이메일/PO/Invoice/BL 번호
- 고급 필터: 날짜, 국가, 상태, 담당자

### 표준 리포트
- 파이프라인 현황(주문 단계·금액)
- 출고/선적 실적(기간/제품/지역)
- 회의 메모 액션 이행률

### 대시보드 권장 위젯
- 나의 오픈 주문/미결 액션
- 월간 출고/매출 트렌드(전년 동기 대비)
- 리스크 알림(납기·신용·클레임)


## 섹션 7: 권한, 데이터 품질, 보안

- 역할 기반 권한(RBAC): 보기/작성/승인 권한 분리
- 개인정보(연락처) 최소 수집·마스킹 정책 준수
- 데이터 품질 점검: 월 1회 중복/누락/이상치 리포트
- 변경 이력: 감사 목적의 필드 이력 추적


## 섹션 8: 단축키·입력 규칙 치트시트

### 입력 규칙(샘플)
- 영문 대문자 코드(예: KR, US), 날짜는 YYYY-MM-DD
- 금액·수량은 자리수/단위 일관(예: 1,234.56; MT)

### 자주 쓰는 키/패턴
- 전역 검색: `CTRL + /` (웹일 경우 브라우저 단축키 주의)
- 필터 토글: 화면 상단 [Filter] 아이콘


## 섹션 9: FAQ (전환·데이터·권한)

Q1. 거래선이 중복으로 보입니다.
- A. 생성 전 중복 검색 필수. 이미 존재 시 담당자 변경/병합 요청.

Q2. 외부 연락처의 개인정보가 수정되지 않습니다.
- A. 권한 제한 가능. 데이터 관리자에게 변경 요청.

Q3. 선적 문서가 주문과 연결되지 않습니다.
- A. 주문번호/출고번호의 정확한 연결 필요. 참조키 확인.

Q4. 영문/국문 표기가 섞여 혼란스럽습니다.
- A. 표시 언어는 사용자 설정 기반. 거래선명은 (국문/영문) 모두 보관 권장.


## 섹션 10: 문제 해결 가이드

- **로그인 불가**: SSO 상태/암호 만료 확인, 캐시 삭제 후 재시도
- **검색 미노출**: 인덱싱 지연 가능(최대 수 분). 고급 필터로 범위 확장
- **첨부 불가**: 파일 유형/용량 제한 확인, 대체 경로(SharePoint 링크) 고려
- **권한 오류**: 역할 변경/이동 시 권한 동기화 요청
